# 电影推荐系统 python31

## 项目概述

本项目是基于Spark实现的的电影推荐系统，集成了管理系统，通过大数据技术为用户提供个性化的电影推荐服务。

## 技术栈

- 后端：Python + Spark
- 前端：Vue

## 功能模块

### 推荐系统

- 实时推荐
- 离线推荐
- 协同过滤推荐
- 统计推荐

### 用户管理

- 历史评分
- 评分统计
- 评分管理

### 数据管理

- 用户管理
- 数据管理

## 运行环境

- Spark集群
- Python环境
- MySQL数据库

## 系统角色

- 系统管理员：负责整个系统的维护与管理
- 普通用户：可以查看推荐电影，进行评分等操作

## 目录结构

```plaintext
/
├── backend                # 后端目录
│   ├── main.py            # 程序主入口
│   ├── recommend          # 推荐模块
│   └── ...
└── frontend               # 前端目录
    ├── index.html         # 主页
    ├── src                # Vue源代码
    └── ...
```

## 核心亮点

- 结合实时与离线推荐，提高推荐的准确性。
- 采用协同过滤技术，实现用户个性化的电影推荐。
- 利用大数据技术Spark，处理海量数据，提高系统性能。
